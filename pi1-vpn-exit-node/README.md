# <img src="https://cdn.simpleicons.org/raspberrypi/A22846" height="22" align="center" /> Pi 1 — VPN Exit Node & Personal Cloud

See [guide.md](./guide.md) for the complete setup guide.

**What this node does:**
- Routes all your devices' internet traffic through Mullvad VPN (Swiss exit IP) via Tailscale
- Runs Nextcloud as a private replacement for iCloud / OneDrive / Google Drive
- Serves as the Tailscale exit node and subnet router for your LAN

**Services:** Mullvad VPN · Tailscale · Nextcloud · Nginx · MariaDB · Docker
